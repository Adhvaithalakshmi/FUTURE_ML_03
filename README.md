# FUTURE INTERNS – TASK 3  
## Resume / Candidate Screening System

### 📌 Task Objective
To build a machine learning system that automatically screens and ranks resumes based on a given job role.

### 🛠 Tools & Technologies Used
- Python
- Jupyter Notebook
- Pandas, NumPy
- NLTK / spaCy
- Scikit-learn

### 📂 Files Included
- task3.ipynb – Complete implementation
- Resume dataset
- Output results

### 🔍 Key Features
- Resume text cleaning and parsing  
- Skill extraction from resumes  
- Matching resumes with job descriptions  
- Candidate ranking based on role fit  
- Skill gap identification
